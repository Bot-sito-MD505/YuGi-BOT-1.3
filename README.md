#YuGi-BOT MD 1.2
Pasos de instalación en termux 

1. Clona el repositorio:

    ```sh
    git clone https://github.com/Eliasar54/YuGi-BOT-1.2.git
    ```
2. navega al ek bot
   ```sh
   cd YuGi-BOT-1.2
   ```     

3. Desconprime los archivos:

    ```sh
    unzip '*.zip'
    ```
5. [Opcional pero recomendable]:

    ```sh
    rm *.zip
    ```

6. Descarga los módulos:

    ```sh
    npm install
    ```

7. Iniciar el bot:

    ```sh
    npm start
    ```
