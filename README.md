#YuGi-BOT MD 1.2
Pasos de instalación en termux 

1. Clona el repositorio:

    ```sh
    git clone https://github.com/Eliasar54/YuGi-BOT-1.2.git
cd YuGi-BOT-1.2
    ```

2. Desconprime los archivos:

    ```sh
    unzip '*.zip'
    ```
4. [Opcional pero recomendable]:

    ```sh
    rm *.zip
    ```

5. Descarga los módulos:

    ```sh
    npm install
    ```

6. Iniciar el bot:

    ```sh
    npm start
    ```
